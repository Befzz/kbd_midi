First of all. It's only 32-bit.

0. Install Microsoft Visual C++ 2010 Redistributable Package (x86)
 ( https://www.microsoft.com/en-US/download/details.aspx?id=5555 )

1. Install MIDI Yoke (It's a virtual cable between my program and other programs that have midi input's)
   ( http://www.midiox.com/myoke.htm#Download )
   ( MidiYokeSetup.msi for WinXP and later )

1a. Run it with capability for WinXP ("prev. windows")

//for test
3. Install Pianoteq Trial version ( https://www.pianoteq.com/try )

CONTROLS::

NUMPAD + / -   -- octave shift
F8 - exit
F9 - random key
F1 - show scan_codes (needed for midi_pc101.txt)
Spacebar - Sustain Pedal

File midi_pc101.txt:
SCAN_CODE MIDI_NOTE
SCAN_CODE MIDI_NOTE
etc....
